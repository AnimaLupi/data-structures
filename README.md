# Data Structures
Fast, light and hassle-free JavaScript data structures, written in CoffeeScript.
- Binary Search Tree
- Graph
- Linked List
- Hash Map
- Queue
- Trie

Check out the wiki page for roadmap, changelog, etc.
## Installation
Using [npm](npmjs.org):
```bash
npm install data-structures
```
## Usage
```coffeescript
Heap = require('data-structures').Heap
heap = new Heap()
```
